Images

Images are integral part of your websites. AdornBox UI provides various customization for your images.

Square images

Square images have default 150px X 150px dimension. You can use img tag with au-image class for square images.

Circular images

Circular images have default 150px X 150px dimension. You can use img tag with au-image-circular class for square images.

Responsive full size images

Responsive full size images have default max container width. You can use img tag with au-image-full-width class for responsive images.

You need to resize your browser window to see responsiveness
